INTERNAL RESORT SECURITY PLATFORM

AI-Driven Zero Trust Guest Data Protection System

An internal cybersecurity platform inspired by the MGM Resorts guest data exposure case. The system protects sensitive guest information using risk-based login, fragmented data storage, honeypot traps, Isolation Forest AI anomaly detection, and real-time security monitoring.

Enter Internal System

Detect

AI behavioral analytics identify abnormal access and insider-threat patterns.

Prevent

Risk-based authentication, CAPTCHA, and account lockout block suspicious access.

Protect

Guest data is split across fragmented MongoDB collections to reduce breach impact.

Respond

Security alerts, honeypot traps, and automated lockouts support fast incident response.
